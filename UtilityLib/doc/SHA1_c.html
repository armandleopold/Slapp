INTRODUCTION

      SHA1 module provide a single function (SHA1Digest) to produce a fingerprint
      of an arbitrary memory chunk. Should be less prone to collision than MD5.

      Original source from Steve Reid.

NAME

      SHA1Digest - compute SHA1 fingerprint of given memory chunk

SYNOPSIS

      void SHA1Digest(void * buffer, int len, void * resblock);

INPUT

      - buffer : source bytes used to compute fingerprint.
      - len    : length in bytes of buffer. If < 0, strlen will be used.

RESULT

      - resblock : must be at least 41 bytes length. will contain the SHA1 sum
               encoded in ASCII and null-terminated. If buffer is NULL, result
               will be an empty string.
